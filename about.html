About

The Soundmap Artist Guesser Helper is a small web utility designed to speed up and simplify the process of narrowing artist candidates for the Soundmap Artist Guesser mini‑game. It lets you add hints (genre, country code, debut year, popularity range, members and gender) and shows matching artists from a the official and updated dataset. The tool is intentionally simple and lightweight: it is meant to assist players by reducing manual filtering and guesswork.

How it works

  1. Add one or more hints using the "Add Hint" button. You can get hints by watching ads on Soundmap.
  2. Select the hint type (Genre, Country Code, Debut Year, Popularity, Members, Gender) and a value.
  3. The results list updates live and shows a "Best Match" candidate based on simple scoring and averages.
  4. Use the Guess feature to test specific artists and further filter candidates.

Why this is useful

The helper reduces time spent scanning large artist spreadsheets or browsing artists. It is not a game client and does not interact with Soundmap servers; it only uses a local data file to produce deterministic and accurate results.
